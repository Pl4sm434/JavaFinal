# Java Game Capstone Project

A Java-based capstone project built using object-oriented programming principles. This project simulates a game environment with multiple interacting components such as players, levels, items, inventory, hazards, and projectiles.

---

## Features

- Player movement and game logic
- Level and platform handling
- Inventory and item system
- Hazards such as lava and moving lava
- Checkpoints and health gates
- Modular multi-class Java structure

---

## Tech Stack

- Java

---

## Project Structure

- `Game.java` – Main game logic and execution
- `Player.java` – Player behavior and movement
- `Level.java` – Level design and management
- `Inventory.java` – Inventory system
- `Item.java` – Item handling
- `Projectile.java` – Projectile mechanics
- `resources/` – Game assets and supporting files
